# Version Runtime Pack V7 - First Principles

1. **Deterministic execution over conversational memory**: every step must be recoverable from files, branch state, commit history, tags, and ledgers.
2. **Roadmap-first traceability**: every contractual roadmap item gets explicit task coverage by default; grouping requires HITL and must preserve individual classification.
3. **Product-quality-first closure**: real product bugs from smoke/E2E tests block closure until fixed and revalidated.
4. **Runtime consistency as a release invariant**: progress, ledgers, commits, tags, validation, audits, smoke, and freeze state must agree before any merge/push/tag gate.
5. **Controlled autonomy**: Codex should automate deterministic engineering workflow, but merge, push, tag, freeze, Runtime Evolution, Product Bug acceptance, and fix approvals remain human-controlled.
