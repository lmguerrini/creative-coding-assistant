# Future Work

These directions extend the current product without implying committed delivery
dates or implemented capability.

## Evaluation and multimodal evidence

The seven-case benchmark should grow through a new, preserved dataset version
covering more supported tasks and domains. Repeated evaluator runs would expose
variance instead of treating one run as definitive.

Multimodal evidence should compare matched text-only and image-plus-text
requests over a synthetic, non-sensitive image set. Blinded criteria for
palette, composition, and motif transfer would distinguish image transport from
measurable image influence.

Controlled model and generation-parameter comparisons could preserve
configuration, latency, usage, and benchmark evidence instead of treating
adjustable controls as validated optimization.

## Professional-tool continuation

TouchDesigner, Houdini, Blender, Unreal Engine, and Unity are currently
code/export targets rather than integrated execution environments. Current
project bundles already provide a structured handoff foundation; deeper
continuation would add explicit target/runtime versions, asset mapping, import
validation, and bidirectional import/export contracts.

Longer-term round-trip workflows could return target-tool changes, diagnostics,
and rendered evidence to CCA for continued inspection and refinement. That
requires explicit compatibility, provenance, conflict, and trust boundaries;
it is not a current capability.

## Preview and runtime coverage

A runtime belongs in the live-preview surface only when it has an equivalent
source policy, sandbox model, failure contract, visible-output test, reload
behavior, and unsupported boundary. Broader format support without those
controls remains code/export-only.

## Hosted operation

A hosted service would require identity, authorization, rate limiting,
encrypted secret management, retention and deletion controls, backups,
dependency monitoring, abuse safeguards, redacted observability, deployment
automation, and a supported browser matrix. The current product remains a local
single-user workstation until those controls exist.
