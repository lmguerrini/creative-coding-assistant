# Codex First Task Prompt

Please inspect this repository and do the following before writing feature code:

1. Summarize the current structure and identify missing foundation pieces.
2. Propose the smallest safe bootstrap plan for this project.
3. Create or refine the repository skeleton to match the architecture docs in /docs.
4. Do not implement product features yet beyond bootstrap.
5. Work on a dedicated bootstrap branch.
6. Keep commits small and focused.
7. Report:
   - plan
   - files changed
   - commits made
   - remaining bootstrap gaps
