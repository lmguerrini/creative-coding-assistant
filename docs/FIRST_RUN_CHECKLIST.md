# First Run Checklist

## Before opening Codex
- Create the repo locally
- Put this starter pack into the repo root
- Initialize git if needed
- Open the repo once in your editor
- Make sure Python environment is available

## Before first task
- Read the docs in /docs
- Decide whether you want project-level Codex config in .codex/config.toml
- Prepare your first branch name
- Start with bootstrap only, not product features

## During the first Codex session
- Ask for a plan first
- Confirm the bootstrap branch name
- Let Codex scaffold the repo skeleton
- Review the diff before accepting larger edits
